Independence Day

Interactive E-Card

Background

Independence Day E-Card

Tools Used: HTML | CSS | JavaScript

One of the major projects during second term at BCIT was creating a holiday E-card that teaches 10 interesting facts using various interactions. The entire process took 1 month.

Process

Step 1 - Creating Assets

After doing a thorough research and laying out the overall composition, creating vector assets were fun and rewarding were done. Each asset, especially the presidents, were carefully created with details. Gradients helped me accomplish the desired looks.

Step 2 - Coding

The most challenging part was positioning and sizing the assets. Once they were placed in the right spots, functions for interactions were implemented using Vanilla JavaScript and CSS Animation All the facts were stored in an array.

Step 3 - Improving

I came up with new ideas. The first one was displaying a halo above the president's head after being clicked to indicate the death. Clouds transformed into hotdogs on a click-event. The fluttering effect was applied on the flag using steps() animation. Finally, Youtube video was imbedded on the map for an extra information.

Reflections

Result

The overall result was quite satisfying as this project resulted in a perfect mark. The components -- research, environment design, implementation using HTML, SVG, CSS -- were successfully utilized to meet the requirements.

What I learned

As aforementioned, it was a great experience to play around with both CSS Animation and JavaScript. By interacting with SVGs, I have learned a new CSS property called, pointer-events.

Next Steps

By using experiences from this project, I will build a responsive E-card that is more intuitive, complex, and adventurous by utilizing SASS and ES6 functions.
