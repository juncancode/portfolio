Recyclone

Educational Web Application

Background

Educating How to Recycle

Tools Used: HTML | CSS | JavaScript

Recyclone was the combined final project for 4 courses in my second term at BCIT. I was responsible for designing and coding while my partner was in charge of User Experience, Style Guides, and creating a persona. Our goal was to create a web-based application that educates children and parents how to recycle most commonly-used items in an interactive way.

Process

Step 1 - UX/UI & Style Guide

The essential part of creating any type of application is researching and style guide as they guide us smoothly to the final product without having to make erratic changes in-between. During these stages, knowing what to do and what not to do helped us achieve the desired concept.

Step 2 - Coding

After creating all the necessary assets in Illustrator, I used HTML, CSS, and JavaScript to enable functions and interactions for the users. Making a responsive landing page that remembers the user's name using local storage was a challenging part of the process.

Step 3 -

One of the essential components was to implement a way to test the users' knowledge based on what they learned through our application. Making a quiz was a common approach and we used cartoony images to engage in a friendly way as our target audience is children and parents.

Reflections

Result

While we received a positive feedback for the imagery and design of the application, the room for an improvement on the functionality -- tutorial and quiz -- was mentioned as they appeared to be cluttered and the initial plan to create a game using bins in the background did not work out due to our limited JavaScript skill, so we went with a plan B, which was a simple quiz.

What I learned

Throughout this project, I encountered numerous problems related to both CSS and JavaScript. This experience made me realize that a problem-solving skill through trials and errors is what makes developers grow. Posting questions and discussing on Stack Overflow helped me tremendously.

Next Steps

The application is playable in a laptop-sized screen or bigger as the only landing page was responsive. I will gradually improve my skills in responsive design and JavaScript in order to create a full-fledged responsive gaming project next time.
